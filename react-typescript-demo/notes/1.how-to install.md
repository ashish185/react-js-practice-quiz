## How to install?
```cmd
npx create-react-app react-typescript-demo --template typescript
```

**This will generate the folder structure like this**
```
└── 📁react-typescript-demo
    └── 📁src
        └── App.css
        └── App.test.tsx
        └── App.tsx
        └── index.css
        └── index.tsx
        └── logo.svg
        └── reportWebVitals.ts
        └── setupTests.ts
    └── 📁public
        └── favicon.ico
        └── index.html
        └── logo192.png
        └── logo512.png
        └── manifest.json
        └── robots.txt
    └── .gitignore
    └── package-lock.json
    └── package.json
    └── README.md
```
