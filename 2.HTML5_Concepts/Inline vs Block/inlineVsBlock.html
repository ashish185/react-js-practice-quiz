Inline vs Block elements

Inline Element:

1. Take width of content. 2. Cannot set height, width and margin top, bottom 3. span, button, input, img,label 4. img, video tag is inline still we can use height and width on that it is an exception.

Block element:

1. Take whole width of parent. 2. Here we can set height, width and margin top, bottom 3. Ex: " div, p, h1 , ul, ol, li, and table"
